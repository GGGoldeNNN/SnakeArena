## 决策空间与心流（来自《体验引擎》）

### 决策范围
每单位时间内给玩家提供的决策量。决策密度直接影响心流体验。

- 决策太少 → 无聊
- 决策太多 → 焦虑
- 最佳点：持续提供有意义的决策，节奏均匀

### 避免心流断层
玩家心流状态被打断的情况，原因包括：
- 长时间的加载画面
- 冗长的过场动画
- 不清晰的指引让玩家卡关
- 操作延迟

### 避免决策溢出
过多信息同时涌入，导致玩家决策瘫痪。

### 回合制的决策节奏
不同游戏类型有天然的决策节奏差异：
- 实时游戏：快速决策，直觉反应
- 回合制游戏：深思熟虑，策略规划
- 半回合制：两者结合

### 决策变化
不同类型决策交替出现，保持新鲜感：
- 战术决策 vs 战略决策
- 短期决策 vs 长期决策
- 风险型决策 vs 稳健型决策

## 技巧与难度曲线（来自《体验引擎》）

- **弹性挑战**：自动匹配玩家水平的挑战机制
- **训练系统**：帮助玩家提升技巧的教学设计
- **难度变化**：难度应随玩家技能自然递增，避免突然峰值
- **处理失败**：失败应被视为学习过程的一部分

## 关卡设计实操方法论（来自《通关！游戏设计之道》）

### 迪士尼乐园类比

关卡设计可以类比迪士尼乐园的游客体验流程：

```
排队 → 上坡 → 体验 → 高潮 → 下车 → 奖励
```

对应到关卡设计中：
| 阶段 | 设计要点 |
|------|---------|
| **排队（进入关卡）** | 让玩家期待，展示关卡主题 |
| **上坡（上升阶段）** | 逐步增加难度，教玩家新机制 |
| **体验（核心内容）** | 主要玩法内容的展开 |
| **高潮（Boss/挑战）** | 综合运用所学技巧的终极考验 |
| **下车（收尾）** | 难度回落，让玩家回味 |
| **奖励** | 通关奖励，激励继续前进 |

### 教学关五大原则

1. **在做中学**：让玩家通过实际操作学习，而非文字说明
2. **一次教一个**：每次只引入一个新概念/新操作
3. **安全环境**：教学阶段让玩家有安全试错的空间
4. **失败即教学**：失败本身就是学习过程的一部分
5. **可跳过**：为有经验的玩家提供跳过教学的选择

### 6种关卡布局类型

| 布局类型 | 特点 | 代表游戏 | 优劣 |
|---------|------|---------|------|
| **线性** | 单一路径，无分支 | 早期FPS | 易控制节奏，但缺乏自由度 |
| **辐射状** | 中心向四周发散 | 黑魂传火祭祀场 | 探索感强，可能迷路 |
| **网状** | 多点互联互通 | 银河恶魔城 | 高自由度，开发成本大 |
| **分支** | 主路径+分支岔路 | 古墓丽影 | 主线明确+探索奖励 |
| **环形** | 起点终点相连 | 跑跑卡丁车赛道 | 适合竞速/竞技 |
| **开放世界** | 全自由探索 | 塞尔达旷野之息 | 极自由，节奏难控制 |

### Boss战设计原则

**核心原则**：
- **模式识别**：Boss的攻击应该是有规律可循的，玩家通过观察学习应对
- **三击规则**：玩家的有效攻击通常需要命中3-5次（或阶段）才能击败Boss
- **多阶段设计**：Boss在不同血量阶段改变攻击模式，保持新鲜感
- **公平性**：每次失败都应让玩家学到新东西，而非被不可抗力击杀

**7种Boss类型**：

| 类型 | 特点 | 设计要点 |
|------|------|---------|
| 坦克型 | 血厚防高、移动慢 | 弱点是速度/灵活性 |
| 敏捷型 | 高速、多段攻击 | 弱点是防御/血量 |
| 法师型 | 远程、范围攻击 | 近战是突破口 |
| 召唤型 | 召唤小兵 | 本体较弱 |
| 机制型 | 需要特殊方式击败 | 解谜+战斗结合 |
| 巨型Boss | 体型巨大，打部位 | 分阶段破坏 |
| 人形Boss | 与玩家能力相似 | 技巧的终极考验 |

### 机关陷阱4大类型

| 类型 | 示例 | 设计要点 |
|------|------|---------|
| **挤压型** | 压扁机关、夹墙 | 规律性移动，给玩家通过窗口 |
| **穿刺型** | 地刺、尖桩 | 视觉提示要清晰 |
| **坠落型** | 坑洞、断桥 | 死亡后有快速重生的检查点 |
| **环境型** | 岩浆、毒气 | 伤害区域边界要可见 |

**组合陷阱原则**：将不同类型的基础陷阱组合，创造新的挑战。

### 紧张度曲线

每个关卡应设计3-5次紧张起伏：

```
紧张度
  ↑
  │   ╱╲    ╱╲    ╱╲
  │  ╱  ╲  ╱  ╲  ╱  ╲
  │ ╱    ╲╱    ╲╱    ╲
  │╱                      ╲
  └─────────────────────────→ 时间
    入口  小高潮  中场  大高潮  出口
```

- 每个小高潮后要留有放松区域让玩家喘息
- 最终高潮（Boss战）应为整关最高点
- 出口阶段难度回落，让玩家以成就感结束关卡
