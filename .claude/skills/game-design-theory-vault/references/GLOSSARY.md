# 游戏设计术语表（GLOSSARY）

> 跨书籍统一术语定义。随着知识库生长持续扩充。

| 术语 | 英文 | 定义 | 来源书籍 |
|------|------|------|----------|
| 游戏机制 | Mechanic | 引导游戏运行的规则 | 《体验引擎》 |
| 事件 | Event | 机制与玩家交互产生的结果 | 《体验引擎》 |
| 情感触发器 | Emotional Trigger | 能够激发情感的事物或意念 | 《体验引擎》 |
| 人类价值 | Human Value | 对人们很重要且可在多种状态间转换的事物 | 《体验引擎》 |
| 情感错位 | Emotional Misattribution | 人无法准确知道情感的真实原因 | 《体验引擎》 |
| 虚构层 | Fiction Layer | 故事背景和世界观，为机制提供情感意义 | 《体验引擎》 |
| 心流 | Flow | 挑战与技能平衡时的沉浸状态 | 《体验引擎》《游戏设计艺术》 |
| 沉浸 | Immersion | 玩家被吸入游戏世界的感觉 | 《体验引擎》 |
| 优雅 | Elegance | 简单规则产生复杂行为的系统特质 | 《体验引擎》 |
| 浮现故事 | Emergent Story | 由游戏系统和玩家行为自然生成的故事 | 《体验引擎》 |
| 策略退化 | Strategy Degradation | 一种策略明显优于其他时，其他策略形同虚设 | 《体验引擎》 |
| 强化程序 | Reward Schedule | 奖励给予的时机和频率模式 | 《体验引擎》 |
| 灰盒 | Graybox | 用简单几何体代替美术资源进行玩法测试 | 《体验引擎》 |
| 依赖堆栈 | Dependency Stack | 游戏开发中层级的依赖关系 | 《体验引擎》 |
| 透镜 | Lens | 从特定角度审视游戏设计的一组问题 | 《游戏设计艺术》 |
| 四大基本元素 | Four Basic Elements | 机制、故事、美学、技术 | 《游戏设计艺术》 |
| 12个体验通道 | 12 Channels of Experience | 玩家体验的12种输入方式 | 《游戏设计艺术》 |
| 胜任感 | Competence | 感觉自己有能力、有掌控的心理需求 | 《游戏设计艺术》 |
| 自主性 | Autonomy | 感觉自己有选择自由的心理需求 | 《游戏设计艺术》 |
| 关联性 | Relatedness | 感觉与他人有连接的心理需求 | 《游戏设计艺术》 |
| 以玩法为核心的设计 | Playcentric Design | 围绕玩家体验展开的迭代设计方法 | 《游戏设计梦工厂》 |
| 正规元素 | Formal Elements | 玩家、目标、操作、规则、资源、冲突、边界、结果 | 《游戏设计梦工厂》 |
| 戏剧元素 | Dramatic Elements | 挑战、故事预设、角色、叙事、构建世界、情节发展 | 《游戏设计梦工厂》 |
| 纸上原型 | Paper Prototyping | 用纸笔等实物快速模拟游戏玩法 | 《游戏设计梦工厂》 |
| 游戏测试 | Playtesting | 让真实玩家试玩并收集反馈的过程 | 《游戏设计梦工厂》 |
| 3C框架 | 3C Framework | 角色(Character)、镜头(Camera)、操作(Control)三要素 | 《通关！游戏设计之道》 |
| 剪影辨识度 | Silhouette | 角色在剪影下的可识别性，决定游戏可读性 | 《通关！游戏设计之道》 |
| 迪士尼类比 | Disneyland Principle | 将关卡设计类比迪士尼乐园设施的节奏结构 | 《通关！游戏设计之道》 |
| 展示莫告诉 | Show, Don't Tell | 通过玩法教学而非文字说明的核心教学原则 | 《通关！游戏设计之道》 |
| 三击规则 | Three-Hit Rule | Boss 三段式设计：找弱点→攻击→换阶段 | 《通关！游戏设计之道》 |
| 单页说明书 | One-Sheet | 一页纸说清游戏概念的设计文档模板 | 《通关！游戏设计之道》 |
| 一级/二级/三级属性 | Primary/Secondary/Tertiary Stats | 属性层级体系：基础→衍生→战斗 | 《平衡掌控者》 |
| 减法公式 / 乘除法公式 | Subtraction / Multiplication-Division Formula | 伤害计算的两种主流公式类型 | 《平衡掌控者》 |
| 伪随机算法 | PRD (Pseudo Random Distribution) | 让随机分布更均匀的概率算法 | 《平衡掌控者》 |
| 战力公式 | Combat Power Formula | 综合战斗力的加权计算公式 | 《平衡掌控者》 |
| 数值膨胀 | Numeric Inflation | 新内容数值远超旧内容导致失控 | 《平衡掌控者》 |
| 保底机制 | Pity System | 概率系统中保障最低获取得益的机制 | 《平衡掌控者》 |
| 游戏性公式 | Gamability Formula | 游戏性 = (选择 + 反馈) × 节奏 | 《游戏为什么好玩》 |
| 核心循环 | Core Loop | 行动→奖励→成长→新行动的持续循环 | 《游戏为什么好玩》 |
| 有趣 vs 好玩 | Fun vs Enjoyable | 短期惊喜 vs 长期持续吸引的区分 | 《游戏为什么好玩》 |
| 十大底层驱动力 | 10 Core Drives | 征服欲、收集欲、好奇心、社交欲、炫耀欲等 | 《游戏为什么好玩》 |
| 蔡格尼克效应 | Zeigarnik Effect | 人对未完成任务的记忆更深刻的心理效应 | 《游戏为什么好玩》 |
| 损失厌恶 | Loss Aversion | 对损失的恐惧约等于对获得渴望的2倍 | 《游戏为什么好玩》 |

---

> 当同一术语在不同书中有不同定义时，会在此标注各书的定义差异。
